# Windows Security Lab

Este é um projeto pessoal criado para desenvolver habilidades em Segurança da Informação.

## Objetivos

- Aprender redes de computadores
- Praticar com Kali Linux
- Realizar análises de rede
- Desenvolver habilidades em cibersegurança

## Ferramentas

- Windows 11
- Kali Linux
- VirtualBox
- Wireshark
- Nmap
